All About GitHub

  • Q: What is a version control system?
  • A: The process of tracking and managing changes to software code
  • Q: What is Git?
  • A: A distributed version control system that allows developers to track versions of files and collaborate on software projects
  • Q: What is GitHub?
  • A: A web-based platform that uses Git to help developers manage and track changes in their code
  • Q: What is the difference between Git and GitHub?
  • A: Git is the stoftare, whereas GitHub is the program that hosts and displays the software
  • Q: Who started GitHub and how was it started?
  • A: Linus Torvalds created GitHub as a replacement for a different program that he developed
  • Q: What company owns it now?
  • A: Microsoft
  • Q: How much does a GitHub account cost?
  • A: Starts at $4.00 per month
  • Q: What is the Octocat?
  • A: The mascot for GitHub

Git/GitHub terms:

  • repository: A place, building, or receptacle where things are or may be stored
  • commit: An individual change to a file
  • fork: Making a copy of a repository into my own GitHub account
  • push: Command that allows you to upload local repository content to a remote repository
  • pull requests: A feature that allows anyone to request for a change, even if they don't have access to the repository
  • workflows: Configurable automated process that will run one or more jobs
  • issues: Tool used to plan, discuss, and track work
  • raw button: Opens a file in a raw form
  • blame button: Feature designed to determine who made changes to a file
