TERM DEFINITION
Machine Language A computer programming language consisting of binary or hexadecimal instructions which a computer can respond to directly.
Debugging The process of identifying, isolating, and resolving errors or bugs in software programs.
Assembler A program for converting instructions written in low-level symbolic code into machine code.
Assembly Language A low-level language that helps to communicate directly with computer hardware.
Translator A program that translates from one programming language into another.
Compiler A program that converts instructions into a machine-code or lower-level form so that they can be read and executed by a computer.
Interpreter A program that can analyze and execute a program line by line.
Cross-platform software The ability of software applications to operate on multiple operating systems (OS) or hardware platforms with little or no modification.
Natural Language An area of computer science and artificial intelligence that deals with the interaction between computers and humans in natural language.
Formal Language A language designed with a specific syntax and semantics, defined with stringent mathematical precision.
Machine Independence Something that is not tied to or dependent on a specific type of computer or computer hardware.
High-level Programming language Programming languages that are used for writing programs or software that can be understood by humans and computers.
Object-oriented programming (OOP) Languages that use objects in programming.
Scripting language Programming language in which there is no compilation and the execution takes place line by line.

Python Collection Data Structures:
  1. list
  2. tuple
  3. set
  4. dictionary
  1. Declared in other languages which is an ordered collection of data. It is very flexible as the items in a list do not need to be of the same type.
  2. Collection of Python objects much like a list but Tuples are immutable in nature i.e. the elements in the tuple cannot be added or removed once created.
  3. Unordered collection of data that is mutable and does not allow any duplicate element.
  4. Unordered collection of data values, used to store data values like a map.
Function Modular units of code designed to perform specific tasks.
Local Variable A variable declared within a specific scope, such as a function or a code block.
Global Variable Variables that are accessible globally, or everywhere throughout the program.
Source Program Set of human-readable instructions written in a programming language.
Object Program Programming paradigm in computer science that relies on the concept of classes and objects.
