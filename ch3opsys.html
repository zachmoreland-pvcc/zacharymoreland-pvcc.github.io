Operating Systems Terminology

TERM DEFINITION
operating system The software that controls the overall operation of the computer including file management, the user interface, and the program execution environment.
job queue Data structure maintained by job scheduler software containing jobs to run.
FIFO Method used to manage data structures or memory allocation where the first element added is the first one to be removed.
interactive processing Operating system that allows the execution of interactive programs.
time-sharing multiprocessing Operating system design that allows multiple users or processes to concurrently share the same system resources.
application software Software designed to perform a group of coordinated functions, tasks, or activities for the benefit of the user.
system software Type of computer program that is designed to run a computer's hardware and application programs.
utility software Type of software which is used to analyse and maintain a computer.
graphical user interface Visual way for users to interact with a computer or software, using graphical elements like windows, icons, buttons, and menus.
shell Command-line interface that allows users to interact with the operating system by entering commands and executing programs.
kernel Core part of an operating system. It acts as a bridge between software applications and the hardware of a computer.
device driver Small piece of software that allows the operating system and other software to communicate with a piece of hardware.
file manager Allows users to create a new file, and modify and delete the old files present at different locations of the computer system.
memory manager Keeps track of the status of each memory location, either allocated or free.
virtual memory Memory management technique used by operating systems, allowing a computer to temporarily increase the capacity of its main memory.
boot strapping Loading a set of instructions when a computer is first turned on or booted.
ROM Type of computer storage containing non-volatile, permanant data that, normally, can only be read, not written to.
deadlock Occurs when two or more processes are unable to proceed because each is waiting for a resource that another process holds.
spooling Temporary storage of data for use and execution by a device, program, or system.
super user Special user account used for system administration.
multi-factor authentication S process that requires users to provide two or more verification factors to gain access to a resource.
