Item Definition
Network Classifications
  1. LAN
  2. MAN
  3. WAN
  1. Local Area Network normally consists of a collection of computers in a single building or building complex
  2. Metropolitan Area Network is a network of intermediate size, such as one spanning a local community
  3. Wide Area Network links machines over a greater distance, perhaps in neighboring cities or on opposite sides of the world
Network Models
  1. bus network
  2. star network
  1. All machines are connected to a common communication line called a bus
  2. One machines serves as a central focal point to which all the others are connected
Protocols
  1. CSMA/CD Ethernet
  2. CSMA/CD Wifi
  1. Network protocol for carrier transmission with a wired connection - associated with bus networks
  2. Network protocol for carrier transmission connected wirelessly
Combining networks
  1. Repeater
  2. Bridge
  3. Switch
  4. Router
  5. Gateway
  1. Device that passes signals back and fothe between the two original buses
  2. Similar to, but more complex than, a repeater. Like a repeater, it connects two buses, but it does not necessarily pass all messages across the connection. Instead, it looks at the destination address that accompanies each message and forwards a message across the connection only when that message is destined for a computer on the other side
  3. Bridge with multiple connections, allowing it to connect several buses rather than just two
  4. Special purpose computers used to forwarding messages - forms connection between networks to form an internet
  5. The point at which one network is linked to an internet
Client/server Models
  1. p2p: peer-to-peer
  2. distributed system
  3. cloud computing
  1. Model that involves processes that provide service to and receive service from each other
  2. consist of software units that execute as processes on different conputers
  3. latest trend in distributed systems - huge pools of shared computers on the network can be allocated for use by clients as needed
Internet addressing
  1. Tier 1
  2. Tier 2
  3. Tier 3
  1. Very high-speed, high-capacity, international WANs
  2. Tend to be more regional in scope and less potent in their capabilities
  3. Independent internet, sometimes called an intranet, operated by a single authority that is in the business of supplying internet access to individual homes and businesses
IP address
  1. ICANN
  2. DNS
  3. TLD: Top-level domain
  4. Country-code TLDs
  1. Nonprofit corporation established to coordinate the Internet's operation
  2. Internet wide directory system
  3. Suffix following the period in a web-address
  4. Two letter TLDs for each specific country
Internet Protocols
  1. FTP
  2. HTTP
  3. SMTP
  1. application for accessing another computer from a great distance
  2. more powerful application than FTP
  3. defines a way that two computers on the network may interact when transmiting an email message from one host to the other
World Wide Web
  1. URL
  2. Client-side activities
  3. Server-side activities
  1. unique address given to each document
  2. webpage involving animation performed by a client
  3. webpage involving animation performed by a server
Internet four-layer software
  1. Applicaiton layer
  2. Transport layer
  3. Network layer
  4. Link layer
  1. place the message typically originates
  2. used by application layer to send and recieve messages
  3. decides which direction a packet should be sent
  4. recieves and transmits the packets
Internet is:
  1. Packet-switching network
  2. Fault tolerant
  3. Redundant
  1. digital network that transfers data files by breaking them up into smaller units called packets
  2. ability of a system to maintain proper operation despite failures or faults in one or more of its components
  3. more than one way to route data
Cybersecurity
  1. malware
  2. virus
  3. worm
  4. trojan horse
  5. spyware
  6. phishing
  7. DDoS attack
  1. malicious software that could be transferred to the computer itself or attack from a distance
  2. software that infects a computer by inserting itself into programs that alaready reside in the machine
  3. autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers
  4. program that enters a computer system disguised as a desirable program
  5. software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack
  6. technique of obtaining information explicity by simply asking for it
  7. the work of generating the attacks is distributed over many hosts (distributed denial of service)
Protection
  1. firewall
  2. spam filter
  3. proxy server
  4. antivirus software
  5. cryptography
  6. public key encription
  1. primary prevention technique that fiters traffic passing through a point in the network
  2. firewalls designed to block unwanted email
  3. software unit that acts as an intermediary between a client and a server with the goal of shielding the clietn from adverse actions of the server
  4. software used to detect and remove the presence of known viruses and other infections
  5. the study of sending and receiving messages securely in the presence of adversaries
  6. involves techniques by which the encryption and decryption algorithms are known, and one of the keys is known, but the encoded message remains secure
